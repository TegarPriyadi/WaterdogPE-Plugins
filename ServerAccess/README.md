  ServerAccess – WaterdogPE Plugin

**ServerAccess** is a lightweight plugin for **WaterdogPE** that allows players to switch servers using chat commands — without enabling cheats.

Perfect for servers that want a clean, simple, and cheat-free multi-server experience.

## 🎥 Demo Video

https://github.com/user-attachments/assets/3d0a98fe-b488-405c-bc0d-5bd828afda76


## 🗃️ Commands

| Command            | Description                  |
| ------------------ | ---------------------------- |
| `.server <name>` | Connect to a specific server |
| `.list`          | Show available server list   |


## 📦 Downloads

**File:** **[ServerAccess.jar](https://ko-fi.com/s/d0283064e6)**

## 📥 Installation

1. Download **ServerAccess.jar**
2. Put the file into:
   ```
   /plugins/
   ```
3. Restart your WaterdogPE proxy
