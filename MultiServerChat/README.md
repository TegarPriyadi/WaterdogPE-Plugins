## 🚀 Features

✔ Cross-server global chat

✔ Private messaging system

✔ Staff-only communication channel

✔ Global announcement system

✔ Lightweight & optimized for performance

<br>

## 🧩 Commands

| Command                    | Description                      | Access         |
| -------------------------- | -------------------------------- | -------------- |
| `.pm <player> <message>` | Send private message to player   | All Players    |
| `.mod <message>`         | Send message only to moderators  | Moderator Only |
| `.announce <message>`    | Send announcement to all servers | Moderator Only |

## 🛡️ Moderator Features

### 🔹 Staff Chat (`.mod`)

Private communication channel only visible to moderators.

Example:

```
.mod Server will restart in 5 minutes
```

✔ Invisible to normal players

✔ Suitable for admin/staff coordination

## ⚙️ Added Moderator Staff

To add a staff moderator, follow these steps:

1. Go to the plugin folder:

```
/plugins/MultiServerChat/
```

2. Open file:

```
config.yml
```

3. Add player username to the `moderators` list

Example:

```
staff_moderator:
	- TEGARP4
	- Moderator
```

4. Save the file, then **restart the WaterdogPE server**
